## 🚀 Быстрый старт

Откройте `index.html` в браузере — никаких зависимостей, всё работает из коробки.

Для работы видео нужен **HTTP-сервер** (YouTube блокирует embed с `file://`):

```bash
# Python
python -m http.server 8000

# Node.js
npx serve . -p 8000

# PHP
php -S localhost:8000
```

Откройте `http://localhost:8000` — видео заработают.

## 📁 Структура проекта

```
telekanal-novii/
├── index.html          # SPA — единственная страница, роутинг через hash
├── 404.html            # Страница 404 с анимацией
├── css/
│   └── style.css       # Весь стиль (~1200 строк)
├── js/
│   ├── data.js         # Данные: видео, товары, команда, контакты
│   └── app.js          # Роутер, рендер, логика (~500 строк)
└── img/                # Все изображения
```

**Тип приложения:** SPA (Single Page Application) — одна HTML-страница, навигация через `#hash`.

## ⚡ Фичи

### Визуал
- **Кастомный курсор** — прыгающий глазик + вращающееся кольцо
- **TV шум** — SVG noise с лёгким подёргиванием (переключается кнопкой ⊘)
- **Звёзды** — мерцающие точки на фоне
- **Бегущая строка** — "СРОЧНО" с мигалкой
- **Глитч-эффект** на заголовке
- **Бургер-меню** — выезжает слева на мобильных

### Функциональность
- **Видео-плеер** — превью → клик → YouTube embed
- **Модалка товара** — зум фото, заказ с валидацией
- **Таймлайн** — история канала с обратным отсчётом
- **OG-теги** — красивое превью при шаринге ссылки
- **404 страница** — с анимацией и юмором

### UX
- **Плавные переходы** между страницами (fade in/out при смене hash)
- **Индикатор текущего раздела** — активная ссылка подсвечена в навигации
- **Сохранение позиции скролла** — при возврате назад остаётесь на том же месте
- **Scroll-to-top кнопка** — появляется при прокрутке
- **Ленивая загрузка модалок** — создаются только при первом вызове
- **Кэш-бастинг** — CSS/JS версии для избежания кэширования

### Адаптив
- **7 breakpoints**: 1400, 1024, 768, 600, 480, 360, 320
- **Landscape режим** — оптимизация для горизонтального телефона
- **Тач-устройства** — курсор скрыт, увеличены зоны клика

## 🛠 Технологии

- **Vanilla JS** — без фреймворков, ~500 строк
- **CSS** — переменные, анимации, grid, flexbox, ~1200 строк
- **SPA роутинг** — `location.hash` + `hashchange`
- **Lazy load** — модалки создаются при первом вызове
- **YouTube API** — embed через iframe
